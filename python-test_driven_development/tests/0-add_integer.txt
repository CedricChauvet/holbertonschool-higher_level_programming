The ``0-add_integer`` module
============================

Usage
-----

```add_integer(a, b)``` returns the addition of its two arguments. For numbers,
that value is equivalent to using the ``+`` operator.

Examples
--------

>>> add_integer = __import__('0-add_integer').add_integer

>>> add_integer(1, 2)
3

>>> add_integer(100, -2)
98

>>> add_integer(6.3,3)
9
   
>>> add_integer(6,3.1)
9
    
>>> add_integer(2.7)
100

>>> add_integer(1.3,3.1)
4
